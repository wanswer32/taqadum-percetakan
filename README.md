# 🖨️ Taqadum Percetakan Website

Website profesional untuk layanan percetakan **Taqadum Percetakan** dengan desain modern dan responsif.

## ✨ Fitur Utama

- 🎨 **Desain Modern** - Interface yang menarik dengan gradient dan animasi
- 📱 **Fully Responsive** - Sempurna di desktop, tablet, dan mobile
- ⚡ **Fast & Smooth** - Smooth scrolling dan animasi yang halus
- 💬 **WhatsApp Integration** - Form kontak langsung terhubung ke WhatsApp
- 🎯 **SEO Friendly** - Struktur HTML yang semantik
- 🌐 **Multi-bahasa** - Interface dalam Bahasa Indonesia

## 📋 Struktur Website

1. **Header/Navbar** - Navigasi fixed dengan menu responsif
2. **Hero Section** - Banner utama dengan call-to-action
3. **Layanan** - 6 layanan percetakan dengan ikon
4. **Portofolio** - Galeri proyek dengan overlay effect
5. **Kontak** - Form kontak dan informasi bisnis
6. **Footer** - Informasi copyright

## 📦 File-file

- `index.html` - File HTML utama
- `style.css` - Styling dan responsive design
- `script.js` - Interaktivitas dan form handling

## 🚀 Cara Menggunakan

1. Download atau clone repository
2. Buka `index.html` di browser
3. Website siap digunakan!
